# mern-chakra-product-store

> **_NOTE:_**  This is an updated version (Chakra UI v3.x) of https://github.com/burakorkmez/mern-crash-course

## Tech Stack
- MongoDB
- Express.js
- React.js
- Node.js
- Chakra UI v3.x

## How to run 

### For Production

```
cd mern-chakra-product-store
npm run build
npm run start
```

### For Development 

```sh
cd mern-chakra-product-store
npm run dev
cd mern-chakra-product-store/frontend
npm run dev
```
