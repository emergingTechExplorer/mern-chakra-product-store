# mern-chakra-product-store

## Tech Stack
- MongoDB
- Express
- React
- NodeJS
- Chakra UI v3.x

## How to run 

### For Production

```
cd mern-chakra-product-store
npm run build
npm run start
```

### For Development 

```sh
cd mern-chakra-product-store
npm run dev
cd mern-chakra-product-store/frontend
npm run dev
```
